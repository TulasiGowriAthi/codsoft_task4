# Task 4: Arena - Rock, Paper, Scissors Game

An interactive, graphical desktop implementation of the classic Rock-Paper-Scissors game featuring automatic scoring metrics and visual round tracking.

## Features
- **Visual Control Layout:** Interactive macro buttons mapping custom choices.
- **Live Scoreboard Counters:** Real-time updates tracking player vs. computer points over multi-round match histories.
- **Contextual Feedback Banners:** Flashing banner indicators that switch color states natively (Green for Wins, Red for Losses, Gray for Draws).
- **Session Eraser:** Global state manager utility to cleanly reset score statistics.

## Tech Stack
- **Language:** Python 3
- **GUI Framework:** CustomTkinter
- **Logic Packages:** 'random'
